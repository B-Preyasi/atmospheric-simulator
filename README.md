# 🌤️ Atmospheric Simulator

An interactive web application that brings atmospheric visual effects to life through engaging animations. Users can trigger beautiful snowfall and balloon-floating effects with a simple click, creating an immersive and visually appealing experience.

## 🚀 Live Demo

**Application URL:**
https://atmospheric-effect-presenter-1043067732.asia-southeast1.run.app

## 📖 Overview

Atmospheric Simulator is a frontend web application developed using **Google AI Studio** and deployed on **Google Cloud Run**. The project demonstrates AI-assisted application development, interactive UI design, animation handling, and cloud deployment.

The application provides two interactive atmospheric effects:

* ❄️ **Snowflakes** – Medium-sized snowflakes fall from the top of the screen for 5 seconds.
* 🎈 **Balloons** – Medium-sized balloons float upward from the bottom of the screen for 5 seconds.
